# Aiti - Product Management App

**Тестовое задание**

Приложение для управления продуктами, построенное на React + TypeScript + Vite.

## Авторизация

Для входа в приложение используйте следующие учетные данные:

- **Логин:** `emilys`
- **Пароль:** `emilyspass`

## Важно!

Приложение использует API [DummyJSON](https://dummyjson.com/) для получения
данных о продуктах.

⚠️ **Внимание:** DummyJSON работает нестабильно с IP-адресов из России. Если
приложение работает медленно или вообще не загружается, рекомендуется
использовать **VPN** для нормальной работы.
